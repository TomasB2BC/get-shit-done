**Title:** "Fork Installation Verification Checklist"

**Section 1: About This Checklist**
Brief explanation: This checklist verifies a successful migration from upstream GSD to the GSD hybrid fork. Run each check in order after following INSTALL-GUIDE.md. Mark each item PASS or FAIL. All items must PASS for a verified installation.

**Section 2: Pre-Install Snapshot (run BEFORE installing)**
Purpose: Capture the state of the existing project so we can verify preservation after install.

Checks (each as a checkbox with command):
```
[ ] PRE-01: Record existing .planning/ file count
    Command: find <project>/.planning -type f | wc -l
    Record: ___

[ ] PRE-02: Record existing custom agents (non-gsd-prefix)
    Command: ls <project>/.claude/agents/ 2>/dev/null | grep -v '^gsd-'
    Record: ___

[ ] PRE-03: Record CLAUDE.md checksum (if exists)
    Command: md5sum ~/.claude/CLAUDE.md 2>/dev/null || echo "none"
    Record: ___

[ ] PRE-04: Record settings.json hook count
    Command: grep -c "hook" ~/.claude/settings.json 2>/dev/null || echo "0"
    Record: ___

[ ] PRE-05: Record config.json contents
    Command: cat <project>/.planning/config.json
    Record: ___
```

**Section 3: Post-Install Smoke Tests (run AFTER installing)**
Purpose: Verify the fork installed correctly and new commands are available.

Checks:
```
[ ] SMOKE-01: gsd-tools.js has fork-only subcommands
    Command: node ~/.claude/get-shit-done/bin/gsd-tools.js 2>&1 | grep -E "resolve-project|register-project|list-projects"
    Expected: All three subcommands appear in output
    Result: PASS / FAIL

[ ] SMOKE-02: Fork-only command files exist globally
    Command: ls ~/.claude/commands/gsd/auto.md && ls ~/.claude/commands/gsd/register-project.md
    Expected: Both files exist (no "No such file" error)
    Result: PASS / FAIL

[ ] SMOKE-03: No local install shadow (from inside target project)
    Command: cd <project> && ls .claude/commands/gsd/ 2>/dev/null || echo "no local GSD"
    Expected: "no local GSD" (local GSD dirs were removed per INSTALL-GUIDE.md Section 2)
    Result: PASS / FAIL

[ ] SMOKE-04: GSD agent files installed
    Command: ls ~/.claude/agents/gsd-*.md | wc -l
    Expected: 11 files (the standard GSD agent set)
    Result: PASS / FAIL
```

**Section 4: Preservation Verification (run AFTER installing)**
Purpose: Verify existing project artifacts are untouched.

Checks:
```
[ ] PRES-01: .planning/ file count unchanged
    Command: find <project>/.planning -type f | wc -l
    Expected: Same count as PRE-01
    Result: PASS / FAIL

[ ] PRES-02: Custom agents still present
    Command: ls <project>/.claude/agents/ 2>/dev/null | grep -v '^gsd-'
    Expected: Same list as PRE-02 (for global agents, check ~/.claude/agents/)
    Result: PASS / FAIL

[ ] PRES-03: CLAUDE.md unchanged
    Command: md5sum ~/.claude/CLAUDE.md 2>/dev/null || echo "none"
    Expected: Same checksum as PRE-03
    Result: PASS / FAIL

[ ] PRES-04: settings.json hooks preserved (count same or higher, never lower)
    Command: grep -c "hook" ~/.claude/settings.json 2>/dev/null || echo "0"
    Expected: >= PRE-04 value (installer merges, never removes hooks)
    Result: PASS / FAIL

[ ] PRES-05: config.json unchanged
    Command: cat <project>/.planning/config.json
    Expected: Identical to PRE-05 (installer never touches .planning/)
    Result: PASS / FAIL

[ ] PRES-06: .coordination/ directory untouched (if exists)
    Command: ls <project>/.coordination/ 2>/dev/null || echo "no .coordination"
    Expected: Same contents as before install (or "no .coordination")
    Result: PASS / FAIL
```
