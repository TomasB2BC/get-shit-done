  <action>
Add three new functions to gsd-tools.js following the existing patterns (loadConfig, cmdFindPhase, etc.):

**1. resolveProject(cwd, alias)**
- Read `.planning/projects.json` from cwd
- If file doesn't exist: return `{ found: false, alias, planning_dir: null, error: 'No projects registered. Run /gsd:register-project first.' }`
- If alias not found: return `{ found: false, alias, planning_dir: null, error: 'Alias "X" not found. Available: a, b, c' }` (list available aliases)
- If found: resolve relative paths against cwd (the root project dir), normalize to forward slashes (`planningDir.replace(/\\/g, '/')`)
- Return `{ found: true, alias, planning_dir: resolvedAbsolutePath }`

**2. registerProject(cwd, alias, planningDir)**
- Read `.planning/projects.json` (create empty object if file doesn't exist)
- Idempotent: if alias exists with same planning_dir, return `{ registered: true, reason: 'already_registered' }`
- If alias exists with DIFFERENT planning_dir, return `{ registered: false, error: 'Alias "X" already registered to: Y' }`
- Validate the planning_dir exists: `fs.existsSync(path.resolve(cwd, planningDir))` -- return error if not
- Add entry: `{ alias, planning_dir: planningDir, registered: new Date().toISOString() }`
- Write JSON with 2-space indent
- Return `{ registered: true, reason: 'new_registration', alias, planning_dir: planningDir }`

**3. listProjects(cwd)**
- Read `.planning/projects.json`
- If doesn't exist: return `{ projects: [], count: 0 }`
- Return `{ projects: [{ alias, planning_dir, registered }], count: N }`

**4. cmdResolveProject(cwd, alias, raw)**
- Call resolveProject, output result
- Raw: return planning_dir path string (empty string if not found)

**5. cmdRegisterProject(cwd, alias, planningDir, raw)**
- If no alias: derive from `path.basename(cwd)` (directory name)
- If no planningDir: use `path.join('.', '.planning')` (current directory's .planning)
- Validate `.planning/` exists at the target before registering
- Call registerProject, output result
- Raw: return 'registered', 'already_registered', or 'error'

**6. cmdListProjects(cwd, raw)**
- Call listProjects, output result
- Raw: return newline-separated list of aliases (or empty string)

Place these functions in the "Commands" section of gsd-tools.js, before the CLI Router section.
