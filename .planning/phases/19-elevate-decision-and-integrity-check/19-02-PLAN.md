**Step 0: Project Resolution** -- Standard --project flag handling (copy from plan-phase.md Step 0).

**Step 1: Validate Environment** -- Verify .planning/ exists. Detect AGENT_MODE (but note: elevate-decision is always interactive regardless). Parse $ARGUMENTS for initial statement.

**Step 2: Pass 1 -- Seed Understanding**

```
>> Pass 1: Seed Understanding
>> "Where can I learn more?"
```

Sub-steps:
1. If $ARGUMENTS contains an initial statement, use it. Otherwise, prompt: "What decision or architectural idea are you thinking about? (A sentence or two is enough -- I will read the context.)"
2. AI reads the initial statement and produces a MINIMAL draft -- assume as little as possible, leave gaps visible. Format:
```
>> Draft Understanding (gaps marked with [?]):
   [Draft with explicit [?] markers where information is missing]

>> What I need to learn more about:
   - [gap 1]
   - [gap 2]
   - [gap 3]
```
3. Ask: "Where can I learn more? Point me to files, docs, prior conversations, or research. (Or type 'skip' if the draft is sufficient.)"
4. If human points to sources: spawn 1-3 parallel Explore Tasks to read those sources. Each probe writes to `.scratch/elevate/pass1-probe-{N}.md`.
5. Read probe outputs. Produce a GROUNDED second draft:
```
>> Updated Understanding (after reading sources):
   [More complete draft, still showing any remaining [?] gaps]

>> Remaining gaps (if any):
   - [gap]
```
6. Clean up `.scratch/elevate/pass1-probe-*.md`

**Step 3: Pass 2 -- Landscape Integrity Check**

```
>> Pass 2: Landscape Integrity Check
>> "Fix the ground before building on it"
```

Sub-steps:
1. Display: "Before writing this decision into planning docs, let me verify they are accurate."
2. Invoke integrity-check: `SlashCommand("/gsd:integrity-check")`
3. Wait for integrity-check to complete.
4. Display: "Planning docs verified. Continuing with extraction."

If integrity-check finds no gaps: "Planning docs are clean. Continuing."
If integrity-check fixes gaps: "Fixed {N} gaps. Planning docs are now current. Continuing."

**Step 4: Pass 3 -- Deep Dig**

```
>> Pass 3: Deep Dig
>> "Extracting the full shape of the decision"
```

Sub-steps:
1. Based on the seed understanding from Pass 1, generate a set of 5 questions. Each question has 3-4 lettered pre-digested options plus a "write your own" option.

Format:
```
## Deep Dig: Question Set {round}

**Q1: [Question about a specific aspect of the decision]**
  a) [Pre-digested option informed by context -- 1 sentence]
  b) [Pre-digested option -- 1 sentence]
  c) [Pre-digested option -- 1 sentence]
  d) Write your own

**Q2: [Question]**
  a) [Option]
  b) [Option]
  c) [Option]
  d) Write your own

[Q3-Q5 same format]

---
>> Which questions would you like to answer? (e.g., "1a, 2b, 3-custom, 5c" or "all")
>> For any, you can pick a letter OR write a custom response.
>> Type "enough" if you feel the picture is complete.
```

2. Parse the response. Update understanding based on answers.

3. **Recursive branching check:** After processing answers, check if any answer surfaced a SECOND decision. If so:
```
>> I'm hearing a second decision here: "[name of branching decision]"
   a) Explore now (nested extraction -- Passes 3-6 for this branch)
   b) Park as todo (/gsd:add-todo with full context)
   c) Note it and continue (capture for later, don't interrupt)
```
If human selects "a": Enter nested extraction loop (Passes 3-6 for the branch). Mark depth: "[Branch: {child-name}]". When branch completes, display ">> Resuming main decision: {parent-name}" and continue.
If human selects "b": Call `/gsd:add-todo` with context about the branching decision.
If human selects "c": Note in the decision record, continue main flow.

4. **Loop control:** After each question set round:
- If human said "enough" -- proceed to Pass 4
- If AI assesses "full picture" (all [?] gaps resolved, consistent understanding): Display "I think I have the full picture. Here is my current understanding: [summary]. Continue with more questions or move on?" If human confirms, proceed to Pass 4.
- Otherwise: Generate next question set (informed by all prior answers), repeat from sub-step 1.

5. **Maximum rounds:** After 5 question-set rounds without the human saying "enough," proactively suggest: "We have covered significant ground. I recommend moving to boundaries (Pass 4). Continue digging or proceed?"
